## 如何在没有Java的电脑上运行

如果目标电脑没有安装Java，您有两个选择：

### 选项1：安装Java（推荐）
1. 下载并安装Java Runtime Environment (JRE)
2. 下载地址：https://www.java.com/zh-CN/download/
3. 安装后即可运行本软件

### 选项2：使用便携式JRE
1. 下载便携式JRE（JRE Portable）
2. 解压到软件目录的jre文件夹
3. 修改run_jar.bat中的Java路径为jre\bin\javaw.exe

## 注意事项

1. **首次运行**：首次运行时，Windows可能会提示安全警告，点击"允许"即可
2. **防火墙**：如果防火墙阻止程序运行，请允许程序访问网络
3. **杀毒软件**：某些杀毒软件可能会误报，请添加到白名单
4. **文件路径**：软件路径中不要包含特殊字符或中文，建议使用英文路径
5. **权限**：确保软件有读写downloads和logs目录的权限

## 故障排除

### 问题1：双击没反应
**解决方案**：
- 检查是否安装了Java环境
- 运行 `java -version` 命令检查Java版本
- 尝试使用CMD运行查看错误信息

### 问题2：显示"找不到Java"
**解决方案**：
- 安装Java Runtime Environment (JRE)
- 或下载便携式JRE并配置路径

### 问题3：无法下载文件
**解决方案**：
- 检查网络连接
- 检查防火墙设置
- 查看logs目录中的日志文件

### 问题4：界面显示异常
**解决方案**：
- 更新Java版本到最新
- 检查屏幕分辨率设置
- 尝试以管理员身份运行
